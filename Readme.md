# what is this

Get perfect shadows

# installation

`npm i shadowizard --save`

Then...

```
import {shadowizrd} from 'shadowizard';

shadowizard({
    shadow_type:'soft',
    padding:false

})
```

## options

shadowizard supports 2 options, both of which are optional;

- _shadow_type_ - \_hard / soft- (Defaults to soft)

- _padding_ - (Defaults to false )
